Project Genesis

Work in progress CSS framework

What is Project Genesis?

Project Genesis is a css framework meant to be the the starting points for your projects.

Use Your Own styles

Don't be shy, add your own styles.

Made with SASS

Project Genesis is modular through the use of SASS. Can add your own own modules or remove modules you don't need.

Responsive

Responsive through the use of a grid system.
