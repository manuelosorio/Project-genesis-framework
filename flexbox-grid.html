Project Genesis - Flexbox Grid

Usage:

  • Follow normal grid naming conventions
  • Add an extension if you want to move a column to another spot with an extension of 'col-order-{Column spot #}
.col .col-5 .col-order-2
.col .col-3.col-order-3
.col .col-4.col-order-1
